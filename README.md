## Get started

#### Yarn
```bash
yarn
```
```bash
yarn dev
```

#### npm
```bash
npm install
```
```bash
npm run dev
```

### GIT ( Passos para fazer commit seguindo o conventional-commit-logs)


## Scripts

- `dev` : Start the App in **developer mode** at "http://localhost:3000";
- `build` : Build the App for **production** at directory `.next/`;
- `start` : Start the server **on port 3000** at "http://localhost:3000";
